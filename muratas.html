MuRatas

is going to help you find the perfect bicycle for your on Estonian market

5 SHOPS IN ONE PLACE

Muratas is gathering bike prices from 5 bike shops gather

FILTERS

Wide variety of filters, which allows for finding a bike with specific requirements.

WEEKLY UPDATES

The website updates its database weekly to provide users with new and reliable information about prices and available models in the Estonian market.

MOTIVATION

Once I decided to find a road bike that will suit all my requirements. And it was actually a pretty challenging task. We have a lot of different bike stores. And checking all of them is a real pain. Many have very bad UI and filter systems. Finding a required bike takes a lot of time and effort. Eventually, I found the bike, but it would be much easier if we had some kind of webpage like HV(hinnavaatlus for the bikes). Unfortunately, we did not have this kind of service. So I decided to create this kind of service.

INTERESTING FACTS

  • Created own data structure for storing data and querying methods
  • The first project where React and Flask were used together
  • This project had one real user, but when he looked at how much the bikes cost he decided not to buy a new bike, all thanks to me
  • Created the first python package which was published on PyPI
